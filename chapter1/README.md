## Exercise 1.12
> 다음 for 루프에서 하는 일은 무엇이며, sum의 마지막 값은 무엇인가?
```cpp
int sum = 0;
for (int i = -100; i <= 100; ++i)
	sum += i;
```

-100 부터 100가지 더하는 루프이며 sum은 0이다.
